
Setup:
* install Visual Studio 2019, use the x86 Native Tools Command Prompt
* install Synthclipse someplace: https://sourceforge.net/projects/synthclipse/
* install git for windows, with bash (or just use mingw64 from msys2 project)
* make sure 32-bit Python 2.7 is installed and edit Oidos/makedist.sh to point to if it's not in your path
* make sure Rust is installed and in your path you can run cargo from command line
* run 'env.bat' to open a development command prompt, and then run 'setup.bat' from that prompt to build all the dependencies
